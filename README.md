# Vue 3 + Vite

# Steps to Run the Project

1. Clone the project

```
git clone https://github.com/samuel-narvaez/Pokedex-global66.git
```

2. Install the dependencies

```
 npm install
```

3. Run the project

```
 npm run dev
```

4. libraries usage

```

 - vuex
 - vuex-persistedstate
 - axios
 - tailwindcss
 - sweetalert2
 - fontawesome

```
